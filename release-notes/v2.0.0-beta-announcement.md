# OpenClaw Manager v2.0.0 Beta 已发布

我们今天把 `openclaw manager v2.0.0 Beta` 发出来了。  
这版的重点不是堆更多功能，而是把桌面端控制 UI 往**真实可用、可回归、可继续收口**推进。

## 这次主要更新
- 进入 `v2.0.0 Beta` 阶段
- 优化启动和页面切换阶段的卡顿
- 调整诊断/刷新与版本检查链路
- 精简部分页面交互，减少误导入口
- 继续推进多实例/当前实例驱动的一致性

## 适合现在体验的人
- 正在内部测试 OpenClaw 桌面端的同学
- 想提前体验新版控制 UI 的用户
- 愿意帮忙反馈性能、交互和实例管理问题的朋友

## 说明
这还是 **Beta**，不是正式版。  
如果你发现卡顿、页面行为不一致、实例切换异常，欢迎直接反馈，我们会继续收口。
