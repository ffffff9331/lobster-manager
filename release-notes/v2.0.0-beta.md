OpenClaw Manager v2.0.0 Beta

`openclaw manager v2.0.0 Beta` 现已发布。

这是一个 **Beta 预发布版本**。这次重点不是继续堆新功能，而是把桌面端控制 UI 往 **真实可用、可持续回归、可继续收口** 推进。

## 推荐下载
当前发布页按 **三平台各一个推荐包** 收口：
- **macOS**：`.dmg`
- **Windows**：`setup.exe`
- **Linux**：`AppImage`

## 本次重点
- 正式进入 `v2.0.0 Beta` 阶段
- 桌面端管理界面继续围绕真实 Tauri 宿主环境收口
- 重点优化启动与页面切换阶段的卡顿问题
- 统一诊断、刷新与版本检查口径
- 精简部分页面交互，减少误导入口与重复配置
- 持续推进多实例 / 当前实例驱动的一致性

## 主要变化
- **性能方向调整**：按页懒加载，减少启动阶段后台自触发加载
- **版本检查收敛**：诊断 / 刷新链路统一带版本检查，不再保留误导性的独立检查入口
- **页面交互整理**：对话工具、设置、模型管理等页面做了保守瘦身与结构收敛
- **跨平台发布链路补齐**：本次 Beta 已补齐 macOS / Windows / Linux 三个平台发布能力

## 当前适用场景
- 内部测试
- 宿主环境人工回归
- 提前体验新版控制 UI
- 收集性能、实例管理与交互反馈

## 当前说明
- 这仍然是 **Beta 版本**，不是正式版完成口径
- 仍可能存在性能尾项、页面细节问题和局部交互缺口
- 如果你用于稳定生产环境，请先谨慎评估

## 反馈重点
如果你在体验过程中遇到以下问题，欢迎优先反馈：
- 启动慢、切页慢、界面卡顿
- 实例切换后行为不一致
- 诊断 / 刷新 / 状态展示口径不一致
- 某些页面仍存在重复入口或误导性交互

感谢参与测试和反馈。
