OpenClaw Manager v2.0.0 Beta

这是 `openclaw manager` 的 Beta 预发布版本。

## 本次重点
- 正式进入 `v2.0.0 Beta` 阶段
- 桌面端管理界面继续围绕真实 Tauri 宿主环境收口
- 重点优化启动与页面切换时的卡顿问题
- 统一部分诊断、刷新与版本检查口径
- 精简部分页面交互，减少误导入口与重复配置

## 这版值得关注的变化
- **性能方向调整**：按页懒加载，减少启动阶段后台自触发加载
- **版本检查收敛**：诊断/刷新链路统一带版本检查，不再单独加误导性的 manager 检查入口
- **页面整理**：对话工具、设置、模型管理等页面做了保守瘦身与交互收敛
- **实例化主线继续推进**：继续围绕多实例/当前实例驱动做真实回归与补尾

## 适合谁
- 内部测试
- 宿主环境人工回归
- 提前体验新版控制 UI

## 当前说明
- 这还是 **Beta 版本**，不是正式版完成口径
- 仍可能存在性能尾项、页面细节问题和局部交互缺口
- 如果你正在做稳定生产使用，请先谨慎评估

## 已知口径
- 当前 Beta 重点不是继续堆新功能，而是收口体验、性能和实例驱动一致性
- 后续如果仍发现卡顿，优先继续排查 hook 自动触发、页面挂载范围与主线程阻塞

感谢参与测试和反馈。
